St. Wilfred's Primary School

Prepare for handover to ESFA

Carry out final checks and update project information if necessary so the ESFA (Education and Skills Funding Agency) can issue the funding when the project is handed over.
Which fields to update

Make sure the information is correct in the following fields in the ESFA Handover section of KIM:

  • Support grant received
  • Funding agreement on GOV.UK
  • Funding agreement to remote storage
  • Funding agreement copy to the trust
What to write about

It can be helpful for the ESFA if you include information about problems and concerns with the project.

Make sure you add notes about any:

  • funding or financial issues
  • significant changes to the model funding agreements and articles
  • issues relating to governance
  • land issues including tenancies at will, condition of buildings, carve out of children's centre, access disputes and sub-leases
  • falling rolls
  • fluctuation in Key Stage 2 or 4 performance
  • enquires about expanding the age range or any other significant changes (opens in new tab)
  • audit trail for resolved disputes with the local authority or complex
  • schools joining a single academy trust to enable it to become a multi academy trust
