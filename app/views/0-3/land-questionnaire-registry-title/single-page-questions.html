Land questionnaire and land registry title

Have you checked and cleared the Land questionnaire?
Check the list of common problems (opens in new tab).

Have you cleared the Land registry title plans?
Check an official copy for boundary issues and buildings that have been excluded.
Help checking Land registry title plans

Check you have an official copy. It will state its from the Land registry, include a title plan and title number.

Check for boundary issues, the boundary should include all school/trust buildings. If the land has become un-registered then check the site plan.

Has the Trust modification order been cleared?
The land questionnaire will state if a Trust modification order is needed.
Help checking the Trust modification order

You will need to send the Trust modification order to the legal department with a copy of the original deed so that it can be cleared. It can take a number of weeks to hear back from the legal team.

Have you cleared the Direction to transfer?
Only foundation schools may need a direction to transfer, you can check this information on the Get Information About Schools service (opens in new tab).
